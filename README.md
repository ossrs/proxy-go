# SRSX

SRSX is the next generation of SRS.

* [proxy-go](./proxy-go/README.md) An common proxy server for any media servers with RTMP/SRT/HLS/HTTP-FLV and WebRTC/WHIP/WHEP protocols support. Proxy server is used to build the next generation [Origin Cluster](https://ossrs.io/lts/en-us/docs/v7/doc/origin-cluster) for SRS.
